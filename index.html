IG 9 Recent

This single page web application was created by myself for the purpose of finding potential influencer accounts on Instagram for marketing purposes. The application bypasses Instagram's restrictive API and scours on-page JSON data for each user account entered. This spits out data about a users account at a given time frame. Its useful in finding user engagement for each account-- things Internet marketers enjoy. You can then store the data in a MySQL database to compare account growth over a time period.
